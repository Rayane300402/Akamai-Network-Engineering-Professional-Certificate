# Akamai Network Engineering Professional Certificate

This repository contains my coursework, notes, lab exercises, and capstone projects completed as part of the **Akamai Network Engineering Professional Certificate**. The program is designed to build a strong foundation in systems administration, networking, security, scripting, and database management — all essential skill areas for modern IT and cybersecurity roles.

Each folder in this repository corresponds to one course in the program and includes project files, configurations, scripts, reports, and other relevant artifacts.

---

## Program Overview

The certificate consists of five core courses:

1. **Operating Systems Fundamentals**
2. **Networking Fundamentals**
3. **Managing Relational Databases**
4. **Python Scripting Fundamentals**
5. **Introduction to IT Security**

Together, these courses develop practical, job-ready competencies for roles such as:
- Junior Network Engineer  
- IT Support Technician  
- Cybersecurity Analyst (Entry-Level)  
- Systems Administrator (Associate-Level)

---

## Courses

### 1. Operating Systems Fundamentals
Focuses on the essentials of Linux system administration. The work in this section demonstrates hands-on use of the command line, user management, file system navigation, permissions, process control, and basic security configurations. Skills gained here form the basis for server management and troubleshooting in real-world environments.

**Key Skills:**  
Linux administration, shell commands, virtualization basics, file system structure, system monitoring, account management.

---

### 2. Networking Fundamentals
Covers foundational networking concepts, protocols, hardware components, routing, switching, addressing, and network troubleshooting. Work in this section reflects the ability to analyze and configure network communication using standard models and diagnostic tools.

**Key Skills:**  
TCP/IP, OSI model, VPNs, DHCP, routing and switching, wireless networking, network architecture, network troubleshooting.

---

### 3. Managing Relational Databases
Introduces relational database systems and SQL. The included assignments show how to design, query, and manage structured data effectively. Practical exercises involve writing SQL queries, designing schemas, and working with database management systems.

**Key Skills:**  
SQL, relational schema design, data manipulation, database queries, normalization, report generation.

---

### 4. Python Scripting Fundamentals
Covers the use of Python for automation and problem-solving. This repository section includes programs demonstrating scripting logic, functions, working with files, interacting with system resources, and basic data structures.

**Key Skills:**  
Python programming, scripting workflows, data structures, input/output handling, debugging, automation.

---

### 5. Introduction to IT Security
Explores fundamental cybersecurity concepts, including risk assessment, identity and access management, encryption, secure architecture, and incident response. Artifacts in this section demonstrate threat analysis and the application of security principles in network environments.

**Key Skills:**  
Network defense, IAM, encryption fundamentals, endpoint security, vulnerability scanning, security policies, incident response.
